# Reusable Prompts

Parameterized prompt templates for common recurring tasks.

## Usage

1. Copy the template content
2. Replace placeholders (e.g., `{FILE}`, `{COMPONENT}`)
3. Execute directly or save to a dated batch

## Templates

| Prompt | Purpose |
|--------|---------|
| [component_tests.md](./component_tests.md) | Generate Angular component unit tests |
| [service_tests.md](./service_tests.md) | Generate backend service tests |
| [update_docs.md](./update_docs.md) | Update DEVELOPMENT_MATRIX + ROADMAP |
