# Templates

> Standardized formats for tasks, research, and documentation.

## What Goes Here

Markdown templates for consistent document creation. Includes task templates, research templates, README templates, and more.

## How to Use

1. **Copy template**: Start new docs by copying relevant template
2. **Fill placeholders**: Replace `{placeholder}` with actual content
3. **Maintain consistency**: Use templates for all new documents

## Key Templates

- `unified_task.md` - Task README structure
- `subdir_readme.md` - Subdirectory README format
- `research.md` - Research document format
- `DEVELOPMENT_MATRIX.md` - Blank matrix for new projects
