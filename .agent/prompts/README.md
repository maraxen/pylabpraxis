# Prompts

> Shared prompt fragments and batch definitions.

## What Goes Here

Reusable prompt snippets, batch dispatch definitions, and common prompt patterns. Enables DRY prompt engineering across agents.

## How to Use

1. **Include in dispatches**: Reference prompt fragments in agent dispatches
2. **Batch prompts**: Define sequences of prompts for dispatch
3. **Keep atomic**: Each prompt fragment should do one thing well
