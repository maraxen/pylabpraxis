# E2E-VIZ-03: Visual Audit - Data & Playground

## Context

**Routes**: `/app/data/*`, `/app/playground`
**Goal**: Visual quality audit for data viz and REPL

## Focus Areas

### Data Visualization

- Chart rendering quality
- Legend readability
- Axis labels and scales
- Responsive behavior of charts
- Color palette for data series
- Export buttons/controls

### Playground (JupyterLite)

- Editor area clarity
- Output display
- Toolbar accessibility
- Loading states
- Error presentation
- Mobile usability

## Requirements

### Screenshots to Capture

1. `/app/data` - Data dashboard
2. Chart detail views
3. `/app/playground` - Initial state
4. Playground with code
5. Playground with output
6. Error states

### Analysis Criteria

- Chart accessibility (colorblind-safe?)
- Text readability in charts
- REPL editor contrast
- Button visibility

## Acceptance Criteria

- [ ] Screenshots for both features
- [ ] Visual findings documented
- [ ] Report: `visual-audit-data-playground.md`
